Django Project Setup:

STEP 1: Create Project Folder

STEP 2: Create Virtual Environment:
      python -m venv venv
      Activate it: venv\Scripts\activate

STEP 3: Install Django:
      pip install django

STEP 4: Create Django Project:

1. django-admin startproject student_demo core 
    this command will create a project structure like this:
    |django-folder
      |student_demo
        |core
    |venv

   and then run dev server using : python manage.py runserver

Using the above steps we will be creating a basic django project and you should see Django success page.
